Este projeto foi desenvolvido em parceira com Wanderson Xavier para a disciplina Sistemas Multimídia (2023/1) 
na graduação de Ciências da Computação. Este projeto aborda conhecimentos da linguagem css, arquivos JSON (JavaScript Object Notation), 
html, manipulação de imagens e áudio.

O presente jogo possui três fazer cujo objetivo é a busca de John Wick por seu cacho sequestrado por 
três personagens de filmes diferentes: GhostFace (Pânico), Motoqueiro Fantasma e Rambo. 
O Jogador deve chegar ao final de cada mapa matando seus oponentes e capangas recolhendo balas para sua arma 
e por fim resgatando seu cachorro
