Inglês:

  This project was developed in partnership with Wanderson Xavier for the Multimedia Systems course (2023/1) in the Computer 
  Science undergraduate program. This project involves knowledge of CSS, JSON (JavaScript Object Notation) files, HTML, image, and 
  audio manipulation.
  
  The game has three stages where the objective is John Wick's quest to rescue his dog, which has been kidnapped by three 
  characters from different movies: GhostFace (Scream), Ghost Rider, and Rambo. The player must reach the end of each map by 
  killing enemies and henchmen, collecting bullets for their weapon, and ultimately rescuing their dog.
